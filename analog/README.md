# Analog Arduino Programs

This directory contains a collection of Arduino sketches that demonstrate various analog input and output functionalities.

## Programs

### 1. `led_fade.ino`

This program demonstrates how to use `analogWrite()` with Pulse-Width Modulation (PWM) to make an LED smoothly fade in and out.

### 2. `led_brightness_potentiometer.ino`

This sketch shows how to read an analog value from a potentiometer using `analogRead()` and map that value to control the brightness of an LED.

### 3. `automatic_night_light.ino`

This is a program that creates an automatic night light. It reads the ambient light level using a Light Dependent Resistor (LDR) and turns on an LED when the light level drops below a certain threshold.
