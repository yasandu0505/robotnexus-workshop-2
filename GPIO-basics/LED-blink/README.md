# ESP32 LED Blink Project

## Objective
Learn how to control a GPIO pin to blink an LED using ESP32.

## Components Required
- ESP32 board
- LED
- 220 resistor
- Breadboard
- Jumper wires
- USB cable

## GPIO Concept
ESP32 has GPIO (General Purpose Input Output) pins that can be programmed as:

- **INPUT** - read signals
- **OUTPUT** - send signals

Digital signals:

- HIGH = ON (3.3V)
- LOW = OFF (0V)

## Code Structure
- `setup()` - runs once
- `loop()` - runs forever

## What You Learn
- How GPIO pins work
- How to control hardware using code
- Basic embedded programming structure
